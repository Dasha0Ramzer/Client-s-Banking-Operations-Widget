## Разработка
По состоянию на 17.01.2026 в проекте реализованы следующие функции:

- **Функция маскировки номера банковской карты**

Пример работы функции:
```
# входные данные
7000792289606361

# выходные данные
7000 79** **** 6361
```

- **Функция маскировки номера банковского счета**

Пример работы функции:
```
# входные данные
73654108430135874305

# выходные данные
**4305
```

- **Функция, маскирующая данные счета или карты**

Пример работы функции:
```
# входные данные пластиковой карты
Visa Platinum 7000792289606361

# выходные данные
Visa Platinum 7000 79** **** 6361

# входные данные счета
Счет 73654108430135874305

# выходные данные
Счет **4305
```

- **Функция, возвращающая корректный формат даты**

Пример работы функции:
```
# входные данные
"2024-03-11T02:26:18.671407"

# выходные данные
"ДД.ММ.ГГГГ" ("11.03.2024")
```

- **Функция, возвращающая список словарей по указанному значению ключа "state"**

Пример работы функции:
```
# входные данные
[
{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
{'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'},
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'},
{'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}
]

# выходные данные со статусом по умолчанию 'EXECUTED'
[
{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
{'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}

# выходные данные, если вторым аргументов передано 'CANCELED'
```

## Тестирование
Проект использует библиотеку `pytest` для тестирования.
Все основные модули проекта покрыты тестами, что позволяет обеспечивать стабильность
и надежность кода. Тесты находятся в папке `tests`, и их можно запустить командой:
```pytest --cov=src --cov-report=html```

### Покрытие тестами
- Покрытие тестами составляет 100%.
- Основные модули, такие как `masks`, `processing`, и `widget`,
имеют высокий уровень покрытия, подтверждающий корректность их работы.
- Отчет о покрытии можно найти в папке `htmlcov`, открыв файл `index.html` в браузере.

### Подробности
- Тесты включают проверку всех критических функций.
- Используются параметризированные тесты и фикстуры для различных сценариев.
